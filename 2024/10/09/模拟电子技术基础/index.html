第六章 运算放大器

6.1 运算放大器介绍

运算放大器有五个端子，两个输入端：同向输入端和反相输入端，一个输出端，还有两个正负直流电源。

理想放大器的特性：无穷大的电压增益、无穷大的输入阻抗、零输出阻抗。

实际上的放大器特性：高电压增益、高输入阻抗、低输出阻抗、宽带宽。

6.2 差分放大器

两个输入端和两个输出端组成。

差分放大器的三种工作状态：单端输入，差分输入、共模输入。

两个发射极电流相等，并且等于总输入端的电流的一般

单端输入：

一个输入端接地，另一个输入端偏置，偏置的输出端信号反向，另一输出端正向。

差分输入：

一个输入端输入同相信号，另一输入端输入反相信号，则输出端输出的都为各自输入端信号反相的增强信号。

共模输入：

两个输入端输入的都为同相信号，则输出端刚好抵消。

共模抑制比：

CMRR = 差分增益与共模增益比值。

以db表示：

CMRR = 20lgCMRR

6.3 运放数据手册中的参数

输入失调电压：

原因是两个放大器的基极-发射极的电压不平衡。

输入失调电压温漂：

运放的输入失调电压越高，温漂则越大。

输入偏置电流：

输入偏置电流为两个输入端电流的平均值。

输入阻抗：

差分输入阻抗为反相输入端和同相输入端之间的总电阻。

差分输入阻抗为两个输入端各自与地之间的阻抗和。

输入失调电流：

两个输入端偏置电流之差。

输出阻抗：

输出端的电阻值。

共模输入电压范围：

运放在正常工作时的电压范围。

开环电压增益：

没有外部器件时，输出电压与输入电压的比值。

转换速率：

输出电压变化所需要的时间。

频率响应：

增益随频率变化的曲线。

其他特性：

  • 短路保护
  • 无闩锁效应：在特定输入条件下，输出不会保持一个状态不变
  • 输入失调调零：通过外部电位器，使得输出电压精确为零

6.4 负反馈

运放对同相输入端信号做出反应，使输出趋于饱和，同时输出的一部分通过反馈通路返回到反相输入端。

作用：

  • 将增益减少到期望值
  • 将输入阻抗控制到期望值和输出阻抗减小到期望值
  • 使得带宽变宽。

6.5 负反馈运放组态

闭环电压增益：

带有负反馈的运放的电压增益。

同相放大器：

闭环增益公式：
Acl(NI) = Rf/Ri + 1 = 1/ B(反馈系数)

电压跟随器是一种特殊的同相放大器，它的输出电压直接与反相端相连，它的闭环增益是1，有高阻抗输入，低阻抗输出的特点。

反相放大器：

同相端接地，电压输出端串联Rf连接到反相端，同时输入电压串联Ri连接反相端。

闭环增益公式：
Acl(I)=-Rf/Ri

6.6 运放阻抗和噪音

同相放大器：

输入阻抗：

在理想状态下，因为反馈电压和输入电压差不多，所以差分电压接近为零，电流也就为零，所以输入阻抗可以看为无穷大。

Zn(NI) = (1+Aol(运放的开环增益)*B) * Zin(运放开环增益时的输入阻抗)

输出阻抗：

在大多数情况下可以把同相放大器的输出阻抗看为0。

Zout(NI) = Zout(运放的开环输出阻抗)/(1+Aol*B)

反相放大器：

Zin(I) = Ri

Zout(I) = Zout / (1+ Aol * B)

第七章 运算放大器响应

运算放大器的截止频率：

该频率比中频增益下降3db。

3db开环带宽：

运算放大器从零到截止频率称为中频，运算放大器在中频时增益稳定，超过截止频率后增益以-20/十倍下降。

单位增益带宽：

增益下降到1（0db）时的频率。

运算放大器可以看成一个电压增益的原件和一个RC滞后网络。

运算放大器的总开环增益：

Aol = Aol(mid)/[ (1+ f^2 /f^2(截止频率) ) ^ (1/2)]

多级放大器每到一个截止频率总下降率就增加-20db/十倍频。

相移：

RC网络会引起输入端到输出端的传输延迟，因此输入信号和输出信号之间会产生相移。

相移 = -arctan(f/fc)

多级放大器的相移响应为相加。

闭环运算放大器的截止频率：

fc(cl) = fc(ol) * (1 + BAol(mid))

B(反馈衰减因子) = Ri / (Ri + Rf)

增益-带宽积：

Acl * fc(cl) = Aol * fc(ol) = 单位-增益带宽

闭环增益的增加会引起带宽的下降，反之亦然。

正反馈：

当从输出端反馈到输入端的信号的相位与输入信号的相位一致时，就会发生正反馈，放大器就会振荡。

环路增益：

运算放大器的开环增益乘以反馈网络的衰减因子

要产生不稳定

  • 必须要有正反馈
  • 闭环放大器的环路增益必须为1

相位裕量：

是使得反馈环的总相移为360时的额外相移。

当相位裕量为正时，则总相移小于360，放大器稳定，反之则不稳定。

只有闭环增益在-20db/十倍频相交时才稳定。

补偿网络：

在级联放大器之间并联电阻串联电容，可以牺牲带宽，并将开环响应向下延长。

第八章 基本运算放大器电路

8.1 比较器

反相输入端接参考电压，同相输入端接输入电压，即为方波发生器。

滞回比较器，又称施密特触发器，同相输入端连接到电阻分压器，使得输出端电压反馈到输入端，输入信号连接到反相输入端。
作用是设置两个参考电压，减少了噪声的影响。

为了使比较器的输出电压能限幅，输出端串联两个互为反向的齐纳二极管，反馈到反相输入端，同相输入端接地，输入电压接反相输入端。

窗口比较器，两个运算放大器并联。上面的放大器的反相输入端接上限幅电压，下面的放大器的同相输入端接下限幅电压，输入电压接到剩余两端。
作用是检测出输入电压在上下两个限幅值之间的变化，超过上限输出高电平，低于下限电压输出低电平。

8.2 求和放大器

一个具有负反馈的放大器，反相端并联若干个输入信号，并串联电阻。

V(out) = - (Rf(反馈电阻)/ R) * (Vin1 + Vin2 + Vin3 …)

用作求和放大器时Rf = R,用作均值放大器时Rf < R,用作比例放大器时根据权重决定电阻大小,R越小权重越大。

8.3 积分器和微分器

积分器：

同相端接地，电压信号通过电阻接到反相端，负反馈是一个电容。

输出电压 = (Ic / C) * t

输出电压变化率 = -Vin / (RC)

微分器：

电阻和电容位置和积分器相反。

输出电压 = I * Rf = - (Vc/t) * Rf *C

8.4 转换器和其他运算放大器电路

恒流源：

稳压源通过输入电阻提供恒定电流，输入信号连接反相端。

电流-电压转换器：

没有输入电阻，输出的电压等于反馈电阻上的电压，该电压与电流成正比。输入信号连接到反相端。

电压-电流转换器：

负载电阻和反相端之间并联一个电阻，然后接地。输入信号连接大同相端。

负载电阻的电流 = 输入电压 / 电阻。

峰值检测器：

检测输入电压的峰值，并将峰值电压储存在电容器。

输入信号通过输入电阻接到同相端，输出端接二极管并接一个电阻回反相端，在二极管和电阻之间并联一个电容并接地，输出电压在中心。

第九章 有源滤波器

9.1 基本滤波器响应

包含一个或多个运算放大器的滤波器叫有源滤波器。

有源滤波器：

  • 低通滤波器
  • 高通滤波器
  • 带通滤波器
  • 带阻滤波器

带通滤波器

BW = fc2 - fc1

f0(中心频率) = (fc2 * fc1) ^ (1/2)

Q = f0 / BW

Q值越高，带宽越窄，选择性越好。

9.2 滤波器的响应特性

有源滤波器的阻尼系数(DF)决定着滤波器呈现哪种特性。

巴特沃斯：

在通带内提供非常平坦的幅值响应，下降率为-20db/十倍频程。

切比雪夫：

通带内有波动，提供更快的下降率。

贝塞尔：

相移随着频率线性增加。

DF = 2 - R1/R2

有源滤波器的基本元件：放大器、滤波器、反馈网络。

有源低通滤波器

单极点滤波器：

f = 1/(2* pi * RC)

Sallen-Key低通滤波器：

最常见的二阶滤波器中的一种，由两个低通RC网络组成，当大于截止频率时它的下降率为-40db/十倍频程。

f = 1/[2 *pi (Ra * Rb * Ca * Cb)^1/2]

级联低通滤波器以获得更高下降率。
